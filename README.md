# C Programming Projects

A collection of C language projects developed as part of college coursework
and independent practice. These projects demonstrate core programming
concepts, memory management, and problem-solving using the C language.

## Topics Covered
- Control flow and loops
- Functions and modular programming
- Pointers and memory management
- Arrays and strings
- File input/output
- Structured program design

## Technologies
- C
- GCC Compiler
- Standard C Libraries

## How to Run
```bash
gcc main.c -o program
./program
